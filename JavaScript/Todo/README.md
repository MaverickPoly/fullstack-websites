# Todo List App in MERN

Simple Todo List Application implemented using MERN Stack

## Technology used
### Backend:
- Express Js
- Mongoose
- JsonWebToken
- CORS
- Bcrypt

### Frontend
- React
- React Router
- Tailwind css
- React Contexts
- Vite
- React Icons


## How to Run?
Both backend and frontend can be run by saying `npm run dev`.
